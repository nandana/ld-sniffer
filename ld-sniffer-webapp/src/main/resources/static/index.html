Linked Data Sniffer

Linked Data Sniffer is a tool for assessing the accessibility of Linked Data resources according to the metrics defined in the Linked Data Quality Model

A list of URIs to be evaluated, one per line.
Timeout in minutes as an integer.
Metrics to be used in the evaluation. By default, all metrics will be included.
