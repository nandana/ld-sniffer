DBpedia Quality Assessment using LDQM

This page shows the results of a quality assessment performed using the Linked Data Quality model (LDQM) and LD Sniffer tool. The resources evaluated include a subset of DBpedia resources (dbo:Person and dbo:Place instances).

In addition to the analysis shown in this page, you can query further details using the SPARQL endpoint or download the RDF dumps from datahub.
